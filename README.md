## Stack
- Next.js 14 (App Router) — JavaScript, no TypeScript
- Supabase (Auth + PostgreSQL + RLS + Storage)
- Vercel (hosting)
- CSS variables, inline styles — no UI libraries

## Setup
Run sql/schema.sql in Supabase SQL Editor first. Then:
npm install
npm run dev
